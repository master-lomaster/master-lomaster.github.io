Что необходимо приобрести для клуба

Инструмент индивидуальный (приобретается при записи в клуб)

  • бокорезы 130-160 мм радиомонтажные

  • зачистка для тонкого провода (до 1мм)

  • линейка метал. 300 мм

  • нож строительный выдвижной (не канцелярский)

  • отвертка крестовая Philips PH1

  • отвертка плоская 160x4 мм

  • перчатки х/б

  • пинцет радимонтажный или медицинский плоский нерж.

  • плата макетная беспаечная 1600 контактов, с проводами

  • плоскогубцы 150 мм тонкие

  • сверла для плат 0.6-2 мм

  • чертилка

Инструмент на стол (комплект на группу из трех чел)

  • держатель для плат с подсветкой

  • кернер, бородок для заклепок

  • ключи гаечные рожковые или комб. рожковый/накидной 6-14мм

  • конструктор металлический винтовой. Общий вес деталей 6-8 кг (несколько наборов, см отд.)

  • круглогубцы радиомонтажные 160 мм

  • маркер лаковый Edding 780 0.8мм

  • метчики М4 с держателем

  • молоток 200 г

  • мультиметр цифровой (самый дешевый не брать!)

  • набор отверток с битами: плоские, Philips (PH0-PH4)

  • надфили набор 6 шт

  • напильник плоский 160 мм

  • пассатижи электромонтажные большие 200 мм

  • паяльник демонтажный с оловоотсосом 220В, 40-50Вт (лучше на 24В для паяльной станции)

  • планшет x86 совместимый с внешней клавиатурой или нетбук (приносить на занятия)

  • полотна для ножевки по металлу

  • сверла 2, 2.5, 3, 3.3, 4, 4.2, 5мм

  • станция паяльная 24В с феном, паяльник со сменными жалами. (можно пока одну на 2 стола)

  • струбцины G-образные 70мм 2шт.

  • тиски 60мм поворотные (крепление струбциной)

  • угольник слесарный 160 мм

  • шестигранник Г-образный (инбусовый ключ) 1.5-6мм

  • штангенциркуль 125 мм

  • щетка-сметка

  • щиток защитный для лица

Инструмент и оборудование общее. Один комплект на клуб

  • весы электронные 2кг, точность 1г

  • зубило узкое и широкое по 2 шт.

  • индикатор рычажный 0-10 мм, погр. 0.01 мм, со стойкой (Есть! но без стойки)

  • инидкатор фазы (отвертка)

  • киянка резиновая 400г

  • ключи гаечные рожковые/накидные и торцевые до 24мм

  • клещи для снятия изоляции с фиксацией, для плоских проводов до 4 мм2

  • краскораспылитель пневматический (пистолет окрасочный)

  • круг войлочный полиров. для наждака

  • кусачки для проволоки, ф до 4мм

  • кюветы пласт и нерж. 1-3л

  • линейка метал. 1000 мм (метр)

  • листогиб роликовый

  • микрометр 0-25 мм

  • метчики и плашки М2-М12 с держателями (Есть набор М3-М5, нужно М6-М12)

  • молоток 400г

  • набор отверток с битами torx TR5-TR50 (звездочки), PZ (pozidriv) PZ0-PZ4

  • набор часовых отверток

  • наковальня малая 5кг

  • напильники плоский, круглый, треугольный, квадратный, насечка №1,2,3 (Есть СССР)

  • ножевка по дереву с мелким зубом (для фанеры)

  • ножевка по металлу

  • ножницы по металлу: прямые и изогнутые

  • обжимка для втулочных наконечников

  • обжимка для телеф./UTP кабеля (Есть!)

  • пистолет клеевой (для термоклея)

  • сверла 2-12мм набор и поштучно

  • совок метал.

  • стакан мерный стекл.

  • термометр электронный до 100С

  • тиски слесарные стальные или чугунные 125мм поворотные

  • тиски станочные (покупаются вместе со сверлильным станком)

  • угольник лекальный 250 мм, кл. 1

  • угольник слесарный 250 мм, кл. 1

  • уровень брусковый сталь. 0.05 мм/м

  • хим. реактивы для изгототовления печатных плат (проявка, травление, лужение, обезжиривание, отмывка)

  • штангенциркуль 250 или 400 мм с часовым или электронным индикатором

  • щетка металлическая

  • щетка-сметка

  • щетка хозяйственная, швабра

  • ящик для мусора пласт. без отверстий 50л

Оборудование электронное / станки

  • генератор сигналов программируемый

  • измеритель RLC малогабаритный, цифровой

  • источник питания лабораторный 3 канала, программируемый (Есть неиспр. см. список)

  • ламинатор двухвальный, вал 320 мм (метал. корпус)

  • коммутатор Ethernet 1Gb/s 8 портов (можно пока не брать, если роутер на 5 портов)

  • компрессор малогабаритный 300вт (для краскораспылителя и сушки/продувки)

  • компьютер (сист. блок) в 19" корпусе 3-4U, (можно desktop мат. плату, лучше серверную)

  • компьютер (сист. блок) Miditower (рабочая станция для САПР CPU 4 ядр. 8Gb min)

  • микроскоп видео USB с держателем для плат

  • монитор LCD 19" и более + клавиатура + мышь (по 3шт)

  • наждак двухдисковый 300-600 Вт

  • осцилограф цифровой 2GSa/s, 2 канала

  • пила монтажная маятниковая дисковая твердосплавная (не абразивная!) или ленточная

  • принтер лазерный или МФУ (желательно, но пока не обязательно)

  • проектор (есть! Cristie F1+ SXGA 1400x1050 250Вт)

  • пылесос промышленный 20л (покупается вместе с фрезерным станком, работает как стружкоотсос)

  • рама экспонирующая, формат А4, CCFL УФ-лампы и светодиоды, с таймером (можно самодельную из светильника или из двух УФ-сушилок для ногтей)

  • роутер Wi-Fi на 5 портов

  • станок сверлильный настольный 2М112 или малогабаритный Корвет 411

  • станок фрезерный ЧПУ, поле 300х400 мм, сервопривод

  • станция паяльная индукционная Metcal/OKI/Thermotronics 500kHz, жала: клин, конус, микроволна, насадки для SOIC, SSOP, QFP, желательно термопинцет
